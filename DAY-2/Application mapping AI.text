| Application                      | Model Type                                             | Primary Metric               | Reason for Selecting the Metric                                                                                                                |
| -------------------------------- | ------------------------------------------------------ | ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Invoice anomaly identification   | Unsupervised Learning (Isolation Forest / Autoencoder) | Precision & Recall           | Detecting fraudulent or unusual invoices is critical. Recall helps capture most anomalies, while precision reduces unnecessary investigations. |
| Audit email spam filtering       | Supervised Learning (Binary Text Classification)       | F1-Score                     | Provides a balance between false alarms and missed spam messages, ensuring reliable classification performance.                                |
| Deal closure prediction          | Supervised Learning (Classification / Regression)      | ROC-AUC / Log Loss           | Evaluates how effectively the model ranks opportunities and estimates probabilities for business decisions.                                    |
| Contract document summarization  | Generative AI (LLM-based Summarization)                | BERTScore & Human Assessment | BERTScore measures semantic similarity, while human validation confirms factual correctness and completeness.                                  |
| Audit report translation         | NLP Sequence-to-Sequence Translation Model             | BLEU Score                   | Assesses translation quality by comparing generated output with reference translations.                                                        |
| Revenue forecast error detection | Regression / Time-Series Forecasting                   | RMSE & MAE                   | Measures prediction deviations and helps evaluate forecasting accuracy in financial reporting.                                                 |
